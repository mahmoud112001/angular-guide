## Overview / Executive Summary

This is an interactive HTML-based guide for learning Angular concepts, designed for the ITI MEARN Track in 2026. The guide covers fundamental and advanced Angular topics with explanations, code examples, and interview-style Q&A. It's a single-page application built with pure HTML, CSS, and JavaScript, featuring a sidebar navigation for easy access to different topics, with a dark theme and modern UI design.

## Key Features

- **Interactive Navigation**: Click on sidebar items to navigate between topics
- **Code Examples**: Syntax-highlighted code blocks with examples
- **Explanations**: Arabic explanations for each concept
- **Interview Q&A**: Common interview questions and answers for each topic
- **Responsive Design**: Works on different screen sizes
- **Dark Theme**: Modern dark color scheme
- **11 Comprehensive Topics**: From basic interpolation to advanced RxJS concepts

## Tech Stack & Requirements

### Technologies Used
- **HTML5**: Structure and content
- **CSS3**: Styling with custom properties, Flexbox, and Grid
- **Vanilla JavaScript**: Interactivity and dynamic content
- **Google Fonts**: JetBrains Mono and Syne fonts

### Requirements
- Modern web browser with ES6+ support
- No server-side dependencies required
- Internet connection for Google Fonts (optional, can be cached)

## Installation & Setup

1. Clone or download the repository
2. Ensure you have a modern web browser installed
3. No additional setup required - it's a standalone HTML file

## Usage & Quick Start

1. Open `angular-guide.html` in any modern web browser
2. Use the sidebar to navigate between different Angular concepts
3. Each topic includes:
   - Explanation of what the concept is
   - Types and variations
   - When and why to use it
   - Code examples
   - Interview questions
